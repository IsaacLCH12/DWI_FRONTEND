# 🏥 Centro de Salud Poquian - Sistema de Gestión Clínica

Este proyecto es una plataforma web Full-Stack diseñada para gestionar reservas de citas médicas, historial clínico y transacciones de pago mediante un flujo de trabajo desacoplado y seguro.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** Angular 17+ (Componentes Standalone)
* **Backend:** Java 17, Spring Boot, Spring Security (JWT)
* **Base de Datos:** MySQL (o PostgreSQL, según tu configuración)
* **Estilos:** SCSS

---

## 🚀 Guía de Instalación y Ejecución Local

Sigue estos pasos para desplegar el proyecto en tu entorno local.

### 1. Requisitos Previos
Asegúrate de tener instalado lo siguiente en tu máquina:
* [Java JDK 17+](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html)
* [Node.js](https://nodejs.org/) (Versión 18 o superior)
* [Angular CLI](https://angular.io/cli) (`npm install -g @angular/cli`)
